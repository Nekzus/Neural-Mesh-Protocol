NMP is a next-generation, high-performance binary transport mesh designed for advanced Artificial Intelligence Agent communication. Conceived as a conceptual and technical evolution of existing context protocols (like **Model Context Protocol**), NMP radically shifts the paradigm from pulling massive data towards **secure Logic-on-Origin execution**.

## Why NMP?

In the rapid evolution of autonomous agents, transferring gigabytes of raw data to central AI nodes for filtering, parsing, or reasoning is increasingly inefficient, slow, and expensive.

<CardGroup cols={2}>
  <Card title="Logic-on-Origin" icon="bolt">
    Execute WebAssembly (WASM) logic exactly where the data lives, eliminating massive data transfers.
  </Card>
  <Card title="Zero-Trust Sandboxing" icon="shield-check">
    Powered by Wasmtime and WASI Preview 1, with microscopic filesystem/network permissions.
  </Card>
  <Card title="High-Performance Transport" icon="rocket">
    Built on Tonic (gRPC) and Protobuf instead of JSON-RPC over stdout, reducing parsing overhead by 90%.
  </Card>
  <Card title="Decentralized Mesh" icon="network-wired">
    Peer-to-peer agent discovery and routing via libp2p Kademlia DHT without central authorities.
  </Card>
</CardGroup>
