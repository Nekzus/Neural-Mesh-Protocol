El Neural Mesh Protocol (NMP) provee a los arquitectos e ingenieros el paquete `@neural-mesh/sdk`, diseñado como una interfaz altamente poderosa pero familiar. Si te encontrabas diseñando servidores para el obsoleto Model Context Protocol (MCP), saltar al hyperespacio arquitectónico de NMP será trivial y prácticamente transparente a nivel código.

En esta guía, programaremos la base matriz de un Nodo NMP Servidor (Data) abriendo su compuerta segura, seguido de su invocador Agente (Client) quien lo penetrará inyectándole inteligencia real.

<Note>
  A nivel subterráneo, la malla NMP compila imperceptiblemente tu lógica al formato compacto WebAssembly para su posterior push e inyección remota. El SDK te abstrae por completo del pesado ensamblador otorgando un dev-experience sublime sin que toques una sola línea que no sea Javascript de negocio.
</Note>

## 2. Iniciar el Gateway (Data Node / Servidor)

Un servidor NMP opera como el Gate-Host. Escucha de fondo inyecciones entrantes que navegan sobre un mar P2P proveniente de mentes remotas. Asegura recursos vitales para transacciones directas.

Crea un archivo llamado `server.ts`:

```typescript server.ts
import { NmpServer } from '@neural-mesh/sdk';
import { z } from 'zod';

// Inicializa el Enclave Local y su Identidad Pública en el DHT
const server = new NmpServer({
  name: "MiHostLocalDatabase",
  version: "1.0.0"
});

// Registrar e Instanciar Capacidad Transaccional Local
// Importante: La magia Logic-on-Origin dicta que tu función interna
// no viajará a la IA, será la IA quien entre hasta la instancia y
// accione esta memoria en tu red local.
server.tool(
  "evaluar_logs_masivos",
  "Busca el bug a velocidades de micro-segundos directamente sobre la RAM.",
  { path_interno: z.string(), term: z.string() },
  async ({ term }) => {
    console.log(`[Host Sandboxing] WebAssembly Remote Agent Ejecutando Local: ${term}`);
    
    // Procesa petabytes sin que salgan en forma JSON al router exterior.
    return {
      content: [{ type: "text", text: `La fuga de memoria sucede en el thread: ${term} de la tabla` }]
    };
  }
);

// Puesta en Marcha hacia el Stream y Anclaje P2P Global.
async function main() {
  await server.connect();
  console.log("[Node] Gateway Despierto. Escuchando Mesh Protocol...");
}

main().catch(console.error);
```

## 3. Despertar la IA (Agente Cliente / Nodo NMP)

Un Cliente en el ecosistema NMP no es un espectador; es Inteligencia Viva. Averigua las topologías expuestas hacia la red pública mediante P2P, y emite secuencias (Payloads) comprimidas matemáticamente con propósitos (Intention).

Forja tu `client.ts`:

```typescript client.ts
import { NmpClient } from '@neural-mesh/sdk';

async function main() {
  // Bautizamos nuestro inyector cognitivo
  const client = new NmpClient({
    name: "Cerebro_Inspector_AWS",
    version: "3.2.1"
  });

  await client.connect();
  console.log("Acoplado al Torrente P2P...");

  // Identificar Host y Ejecutar Extracción Dirigida.
  try {
    const receipt = await client.callTool("MiHostLocalDatabase", "evaluar_logs_masivos", {
      path_interno: "/var/log/syslog/",
      term: "SEGFAULT"
    });

    console.log("Transacción Devuelta. Validada en ZK-Hardware:");
    console.log(receipt.content[0].text);
  } catch (error) {
    console.error("Sandboxing Reject - Exec Bloqueado:", error);
  }
}

main().catch(console.error);
```

## 4. Ensamblar e Iniciar Malla (The Mesh)

Lanza tu escudo primero en un terminal asincrónico:
```bash
npx tsx server.ts
# Console: [Node] Gateway Despierto. Escuchando Mesh Protocol...
```

Luego, clona y ensambla un segundo thread para ejecutar tu ente local:
```bash
npx tsx client.ts
# Console: Acoplado al Torrente P2P...
# Console: Transacción Devuelta. Validada en ZK-Hardware:
# Console: La fuga de memoria sucede en el thread: SEGFAULT de la tabla
```
