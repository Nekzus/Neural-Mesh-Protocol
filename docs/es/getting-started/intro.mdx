NMP es una red de transporte binario de alto rendimiento diseñada específicamente para la comunicación avanzada entre Agentes de Inteligencia Artificial. Concebido como la evolución técnica de los protocolos de contexto (como el **Model Context Protocol**), NMP cambia el paradigma de la transferencia masiva de datos hacia la **ejecución segura de Logic-on-Origin** (Lógica en el Origen).

Si MCP es el "puerto USB-C para aplicaciones de IA", **NMP es el sistema nervioso de fibra óptica** que habilita una verdadera inteligencia descentralizada.

## ¿Por qué NMP?

En las arquitecturas tradicionales, transferir gigabytes de datos (logs, bases de datos) para que un LLM los analice es ineficiente y costoso en términos de latencia y tokens. NMP invierte la arquitectura: **llevamos el proceso a los datos, no los datos al proceso.**

## ¿Para quién es NMP?

*   **Desarrolladores de Aplicaciones IA:** Permite analizar repositorios o bases de datos remotas al instante sin latencia de red masiva.
*   **Propietarios de Datos y Empresas:** Expón herramientas y datos sensibles de forma segura. Los datos brutos nunca salen de tu red; solo se exporta la respuesta final.
*   **Comunidad de IA:** Un estándar universal y matemáticamente verificable para desplegar misiones de inteligencia autónoma.

## Pilares de la Malla Neuronal

1.  **Seguridad por Capacidades:** Los permisos son descriptores de archivos mapeados estrictamente en el sandbox. Solo se ve lo que se concede.
2.  **Handshakes Post-Cuánticos:** Negociación vía Kyber (ML-KEM-768) para mitigar ataques *Harvest Now, Decrypt Later*, con cifrado AES-256-GCM.
3.  **Watchdogs en Tiempo Real:** Módulos `.wasm` persistentes que "duermen" en el servidor y despiertan para emitir alertas asíncronas solo cuando detectan anomalías.
4.  **Computación Ciega (TEEs):** Preparado para ejecutar el motor `Wasmtime` dentro de AWS Nitro Enclaves para cifrado de memoria a nivel hardware.
5.  **Guardianes AST:** Inspección preventiva de binarios `.wasm` para destruir amenazas antes de la inicialización del sandbox.
