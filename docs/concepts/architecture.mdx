NMP replaces the client-server JSON-RPC pull model with a **decentralized push model**.

## Logic-on-Origin
Instead of an AI downloading large chunks of data to read, NMP pushes an Agent's logic (compiled to WebAssembly) to the Server. Big Data stays where it is.

## libp2p Kademlia DHT
NMP nodes discover each other using a decentralized Distributed Hash Table. There is no central registry, ensuring extreme resilience.
