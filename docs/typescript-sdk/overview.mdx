The `@neural-mesh/sdk` connects Node/TypeScript ecosystems with the high-performance binary NMP network. 

## Features

- **Javy Compilation:** Transparently compiles JS tools into WebAssembly.
- **Zod Schemas:** Define parameters identically to standard MCP servers.
- **NmpMcpBridge:** Run legacy JSON-RPC MCP servers side-by-side with NMP nodes over the mesh.
